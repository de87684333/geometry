# geometry

Librairie minimale pour calculer la surface et le périmètre d'un rectangle.

Installation depuis GitHub :
- Avec pip :
  pip install git+https://github.com/de87684333/geometry.git
- Avec poetry :
  poetry add git+https://github.com/de87684333/geometry.git

Utilisation :

```python
from geometry import area, perimeter

print(area(3, 4))       # 12
print(perimeter(3, 4))  # 14
```
